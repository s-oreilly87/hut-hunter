# Security Notes

If you discover a security issue in Hut Hunter, do not open a public issue with
exploit details.

Please report it privately to the repository owner with:

- a short description of the issue
- affected deployment shape or feature area
- reproduction steps or proof of concept
- suggested mitigation if you have one

Sensitive areas in this project include:

- authentication cookies
- stored booking credentials
- saved occupant details
- noVNC checkout exposure
- background worker browser automation

Until a private reporting channel is added, treat security disclosures as
direct maintainer outreach.
